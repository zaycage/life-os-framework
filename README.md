# Life OS Framework

Life OS is a personal philosophy and systems framework for intentionally designing a life.

The framework is organized into three layers:

1. **Principles** — How I choose to live.
2. **Levers** — What I can intentionally improve.
3. **Outcomes** — What naturally emerges.

Its central philosophy is:

> Build systems. Remove friction. Let consistency compound.

## Repository Structure

- `01-constitution` — Core philosophy and principles
- `02-foundations` — Resources that support the framework
- `03-systems` — Repeatable processes
- `04-domains` — Areas where the framework is applied
- `05-reference` — Templates and supporting material
- `06-examples` — Generalized implementations

## Status

This framework is an evolving work in progress.
